# Stablebooks Day 34 Public Announcement Copy

## Positioning

Stablebooks should be announced as a controlled Arc Testnet receivables MVP.

Use clear testnet language:

```text
controlled Arc Testnet testing
real Arc Testnet USDC payments
not mainnet
not production funds
feedback welcome
```

Avoid these claims:

```text
mainnet ready
production payments
Circle-native event delivery
merchant webhooks enabled by default
audited
fully decentralized
```

## Primary X.com Post

```text
Stablebooks is open for controlled Arc Testnet testing.

It is a lightweight stablecoin receivables workspace:

- create a customer
- issue an invoice
- pay exact Arc Testnet USDC from MetaMask
- watch the invoice finalize with txHash, block, and match evidence

Try it:
https://stablebooks-app.xyz

This is testnet only. Please do not use mainnet funds.

If something breaks, send the invoice ref, txHash, screenshot, wallet, browser,
and approximate time.
```

## Shorter X.com Post

```text
Stablebooks is live for controlled Arc Testnet testing.

Create an invoice, pay exact Arc Testnet USDC from MetaMask, and see the
payment finalize with txHash + block evidence in the operator workspace.

Testnet only:
https://stablebooks-app.xyz

Feedback welcome.
```
