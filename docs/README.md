# Docs

This directory contains the project specifications and bootstrap documents for Stablebooks.

## Structure

- `product/` contains the product strategy, PRD, IA, execution notes, frontend plan, and backend/API plan
- `db-schema.md` contains the canonical Day 2 Postgres schema decisions and invariants
- `day-2-5-storage-hardening-plan.md` contains the next storage-contract hardening steps after Day 2
- `day-3-execution-plan.md` contains the Day 3 canonical payment backend flow plan
- `day-4-execution-plan.md` contains the Day 4 real Arc ingestion adapter plan
- `day-4-arc-source-strategy.md` contains the chosen MVP Arc event source strategy
- `day-4-arc-event-contract.md` contains the canonical Arc inbound event contract and validation rules
- `day-4-provider-assumptions.md` contains provider assumptions and open questions for Slice 2
- `day-4-arc-dev-runbook.md` contains the local runbook for the new Arc dev ingestion path
- `day-4-acceptance-note.md` contains the final Day 4 acceptance checklist and known boundaries
- `day-5-execution-plan.md` contains the Postgres runtime hardening and Arc regression plan
- `day-5-acceptance-note.md` contains the final Day 5 acceptance checklist and known boundaries
- `day-6-execution-plan.md` contains the incremental Postgres runtime migration plan
- `day-6-acceptance-note.md` contains the final Day 6 acceptance checklist and known boundaries
- `day-7-execution-plan.md` contains the first incremental Postgres write-path migration plan
- `repo_bootstrap_plan.md` contains the implementation-first repository setup sequence
